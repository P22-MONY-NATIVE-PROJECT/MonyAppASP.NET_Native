# Bugs Report

## Сторінка логіну/реєстрації
### UI-UX bugs
- коли клікаєш на сховати/показати пароль з відкритою клавіатурою, то клавіатура зникає
- не можливо клікнути на будь-яку кнопку коли відкрита клавіатура

### Functionality bugs
- при реєстрації/вході не завантажується профіль користувача
- при реєстрації/вході не можливо вийти


## Головна сторінка (список категорій)
### UI-UX bugs
- При створені операції клавіатура ховає форму

### Functionality bugs
- на формі створення операції немає валідації Amount (вводжу 1000000000000000 і нічого не відбувається)
- на формі створення операцій можна вводити від'ємний amount
